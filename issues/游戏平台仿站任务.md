## 部署配置

### Cloudflare Pages设置
- 构建命令: `npm run generate`
- 输出目录: `dist`
- 支持静态文件托管
- 自动HTTPS和CDN

### 环境变量
无需特殊环境变量，纯静态站点

## 项目结构
```
sprunky-games/
├── components/          # Vue组件
├── layouts/            # 布局文件
├── pages/              # 页面路由
├── data/               # 数据文件 (重要)
├── composables/        # 组合式函数
├── assets/css/         # 样式文件
├── public/             # 静态资源
└── nuxt.config.ts      # Nuxt配置
```
